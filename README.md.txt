# Student Management System using Python and MySQL

A Python-based Student Management System that performs CRUD (Create, Read, Update, Delete) operations using a MySQL database.

## Features

- Add student records
- View all students
- Search student by ID
- Update student details
- Delete student records
- Input validation and exception handling
- Automatic database and table creation

## Technologies Used

- Python
- MySQL
- mysql-connector-python

## Prerequisites

- Python 3.x
- MySQL Server
- mysql-connector-python

## Installation

Install the required package:

```bash
pip install mysql-connector-python
```

Update the database connection details in the code before running the program.

## How to Run

```bash
python student_management_system.py
```
